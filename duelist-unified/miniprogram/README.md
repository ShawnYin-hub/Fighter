# 决斗家 - 微信小程序

## 使用说明

这是一个使用 web-view 组件加载 H5 页面的微信小程序项目。

## 开发环境设置

1. 确保本地开发服务器运行在 `http://localhost:3000`
2. 在微信开发者工具中打开 `miniprogram` 目录
3. 在项目设置中关闭"不校验合法域名"选项（开发阶段）

## 生产环境部署

1. 将 `dist` 目录部署到服务器（如 Nginx、Vercel、Netlify 等）
2. 修改 `pages/index/index.js` 中的 `webviewUrl` 为实际部署地址
3. 在微信公众平台配置业务域名
4. 重新编译小程序并上传

## 注意事项

- web-view 组件需要配置业务域名
- 本地开发时需要在微信开发者工具中设置"不校验合法域名"
- 生产环境必须使用 HTTPS 协议
